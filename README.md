# MyShopApp
E-Commerce app using flutter.
## Fearures:
- [x] Beautiful UI from the flutter way.  
- [x] Use firbase as backend. 
- [x] Animation.
- [x] Screens:
    * Splash.
    * Sign in.
    * Sign up.
    * OTP.
    * On Boarding.
    * Profile.
    * Home.
    * Cart.
    * Favorite.
    * Products.
    * Product Detail.
